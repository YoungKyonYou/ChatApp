# 백엔드 과제

### 실행 방법
- 서버 시작 전 선실행
  - src/main/resources/docker 안에 docker-compose.yml 실행
    - redis, kafka 도커 이미지로 실행

### 가정
- 과거 메시지는 조회는 무한 스크롤 형식으로 진행
  - 가장 최근 메시지를 10개씩 보여주는 방법으로 채팅방에 스크롤을 올리면 보여주는 걸 가정 (카카오톡과 동일)
